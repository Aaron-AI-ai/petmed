# 중독 위험 음식 (응급)

## 절대 급여 금지 목록

| 음식 | 독성 원인 | 주요 증상 |
|---|---|---|
| 초콜릿, 커피, 카페인 | 메틸잔틴 | 구토, 설사, 헐떡임, 갈증, 부정맥, 떨림, 발작. 다크초콜릿일수록 위험 |
| 자일리톨 (껌, 사탕, 치약 등) | 저혈당 유발, 간 손상 | 구토, 무기력, 운동 실조 → 발작으로 진행 가능 |
| 포도, 건포도 | 주석산(타르타르산) 추정 | 신장 손상 (개는 대사 불가) |
| 양파, 마늘, 부추, 파 | 적혈구 손상 | 빈혈 — 생것, 조리, 분말 모두 위험 (개·고양이 공통) |

## 응급 대처 요령

1. 남은 음식에 대한 접근을 즉시 차단
2. **바로 동물병원 또는 응급센터에 연락** — 섭취량/시간을 알리기
3. 구토, 무기력, 떨림, 허약, 허탈 등의 증상 관찰
4. 빠른 치료가 예후를 크게 좌우함

미국 기준 24시간 상담: Pet Poison Helpline (855-764-7661). 국내는 가까운 24시간 동물병원 연락처를 미리 저장해 둘 것.
