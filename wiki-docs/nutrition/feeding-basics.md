# 사료와 영양 기본

## 연령별 사료 선택

강아지와 고양이 모두 생애 단계에 맞는 사료를 급여한다:

- **성장기**: 퍼피/키튼 전용 사료 (고열량, 성장 영양소)
- **성견/성묘**: 유지 사료
- **노령**: 노령 전용 사료 (관절, 신장 배려)

## 급여량

- 사료 포장의 권장 급여량을 기준으로 시작
- 체중과 체형(BCS) 변화를 관찰하며 조절
- 간식은 하루 열량의 10% 이내 권장

## 사람 음식 주의

초콜릿, 포도, 양파, 자일리톨 등은 중독 위험 — 상세 목록은 `emergency/toxic-foods.md` 참고.

## 물

- 신선한 물을 항상 제공
- 고양이는 음수량이 부족하기 쉬워 하부요로질환(FLUTD) 위험과 연관 — 음수대 여러 곳 배치, 습식 사료 병행 고려
